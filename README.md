# 🚀 击败美国佬 - 太空射击游戏

一个基于 Pygame 开发的经典太空射击游戏，玩家控制中国战机击败入侵的美国外星人舰队。

## ✨ 游戏特色

### 🎯 核心玩法
- **经典射击体验** - 控制战机左右移动，发射子弹消灭外星敌人
- **智能敌人生成** - 外星舰队会自动改变移动方向和下降高度  
- **难度递增系统** - 随着关卡提升，游戏速度加快，得分倍数增加

### 🛡️ 护盾系统
- **能量护盾** - 按 `1` 键激活能量护盾保护战机
- **多重充能** - 最多可储存 3 次护盾使用次数
- **冷却机制** - 使用后需要时间充能，策略性使用至关重要
- **视觉特效** - 护盾激活时有动态脉冲效果

### 📊 游戏系统
- **实时统计** - 显示当前分数、最高分和等级
- **生命显示** - 左上角显示剩余战机数量
- **等级提升** - 消灭所有外星人后进入下一关

## 🎮 游戏控制

| 按键 | 功能 |
|------|------|
| **← / →** | 左右移动战机 |
| **空格键** | 发射子弹 |
| **1 键** | 激活护盾 |
| **Q 键** | 退出游戏 |
| **鼠标点击** | 开始游戏/重新开始 |

## 🛠️ 安装与运行

### 环境要求
- Python 3.7+
- Pygame 2.0+

### 安装步骤
```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd alien-invasion

# 安装依赖
pip install pygame

# 运行游戏
python main.py
